## Что я сделал:

Написал PHP скрипт который получает компании из Битрикс24. Сделал с пагинацией чтобы не грузить все данные сразу.

**Что умеет:**
- Подключается к Битрикс24 через вебхук
- Получает компаниями постранично (по 20 на страницу)
- Показывает в удобной таблице
- Если вебхук не указан, показывает примерные данные для теста
- Обрабатывает ошибки подключения
